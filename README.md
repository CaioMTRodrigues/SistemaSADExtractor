# Sistema SAD Extractor

Projeto da Universidade de Pernambuco da disciplina de Sistemas Multimídia, para a Secretaria de Administração de Pernambuco - SAD.

## Arquitetura e Tecnologias

### **Frontend**
- React  
- Vite  
- TypeScript  

### **Backend**
- Node.js + Express  
- Arquitetura em camadas  
- Leitura de PDF com `pdf-parse`  
- Integração com IA via OpenAI / OpenRouter  

### **Banco de Dados**
- PostgreSQL  
- Prisma ORM  

### **Segurança**
- Autenticação via **better-auth**
